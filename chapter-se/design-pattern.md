# 设计模式

如何学习设计模式：要能讲解出来，能举出例子。

分类（以 GoF 书籍为准）：

+ 创建型模式 (creational patterns)
  + Singleton 单例
  + Builder 构建（生成器）
  + Prototype 原型
  + Factory method 工厂方法
  + Abstract factory 抽象工厂
+ 行为型模式 (behavioral patterns)
  + Strategy 策略
  + State 状态
  + Observer 观察者
  + Command 命令
  + Iterator 迭代器
  + Mediator 中介
  + Interpreter 解释器
  + Template method 模板方法
  + Chain of responsibility 责任链
  + Memento 备忘录
  + Visitor 访问者
+ 结构型模式 (structural patterns)
  + Decorator 装饰者
  + Facade 外观
  + Adapter 适配器
  + Proxy 代理
  + Bridge 桥接
  + Flyweight 享元（蝇量）
  + Composite 组合
